# FOFA sdk批量导出ip,port,title,protocol

**不定期更新写入excel、多线程、GPU、多核版本**

Tips:  
	-m fofa邮箱  
	-k APIkey
	-q 查询语法
	-p 页数，一页100条数据
	-o 输出文件名
useage：

```
python fofa-sdk.py -m 123456@gmail.com -k e839b8781c6b60d381562823e1123456 -q /login.rsp -p 3 -o 2018-9995.txt"
```
